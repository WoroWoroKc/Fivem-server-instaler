# Install:

* Aby odpalić skrypt musisz być zalogowany jako root. Możesz to zrobić komędą
```
  sudo su
```

* Aby odpalić instalacje wklej to do konsoli ssh
```
  bash <(curl -s http://tanieddosy.pl/install.sh)
```
